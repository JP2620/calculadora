# calculadora

TP1 de Sistemas de Computación UNC 2021

## Parte 1

Se le pedirá al estudiante implementar una Calculadora
programada en lenguaje C. Cada una de las operaciones
deben estar programadas en lenguaje ASM en archivos
separados, utilizando la pila para convocar, enviar
parámetros y devolver resultados. La Calculadora debe
permitir sumar y restar enteros y binarios, presentar el
resultado de la suma, borrar lo ingresado y finalizar su
ejecución.

## Parte 2

Una de las aplicaciones directas del conocimiento presentado
en la primera parte, está directamente vinculada al área de 
Ciberseguridad. Concretamente a la construcción de exploits y
evaluación de la seguridad de aplicaciones. Se recomienda revisar
los contenidos de la Fundación Sadosky Guía de auto-estudio para
la escritura de exploits. En la Teoría de esta Guía, se revisan
prácticamente los mismos contenidos desarrollados en la primera
parte de este TP.

Luego, en la Práctica Nivel 1 se ven variantes del "Smash the Attack".

El objetivo de esta segunda parte del TP#1 es desarrollar las aptitudes
para evaluar y prevenir problemas de seguridad en la programación
vinculada a las convenciones de llamadas entre lenguajes de diferente
nivel como C y ASM.
